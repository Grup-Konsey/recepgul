Home Page

Home Page Nedir?

Home Page dilimize anasayfa olarak çevirlmiştir. Tarayıcımızı(Google Chrome, Mozilla, Explorer) gördüğümüz ilk sayfa bizim home pagemiz yanı anasayfamızdır. Genellikle homepage olarak bir arama motoru belirlenir.(Google, Yandex)
Anasayfa terimi web siteleri için de kullanılır. Bir web sitesine girdiğimizde genellikle bizi karşılayan ilk sayfa anasayfadır. Web sitesi anasayfa üzerine oluşturulur. Anasayfadaki bağlantılar ile sitenin diğer sayfalarına ulaşırız.
